  • RGBA: Red, Green, Blue, Alpha

  • HEX: Hexadecimal

What is a pseudo class?

A pseudo-class is used to define a special state of an element that we want some sort of styling to apply to. For example, when a user hovers over a button, we can change the color of the text or the background color of the button. The pseudo-class is used to define this special state.

  • :visited
  • :hover
  • :active
